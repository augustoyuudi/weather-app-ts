## Prerequisites

* [npm](https://nodejs.org/en/) / [yarn](https://yarnpkg.com/)

## Project setup

### Installing dependencies
```sh
yarn install
```

### Compiles and hot-reloads for development
```sh
yarn start:local
```

### Compiles and minifies for production
```sh
yarn build
```

### Run unit and functional tests
```sh
yarn test
```
